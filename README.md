# 🖥️ Windows 11 UI Clone

A front-end project that replicates the look and feel of the Windows 11 desktop interface using **HTML, CSS, and JavaScript**.

## 🚀 Live Demo

https://krishna-2013.github.io/Windows-11-UI-Clone/

---

## 📌 Features

* 🪟 Desktop layout similar to Windows 11
* 📊 Taskbar with centered icons
* 🔍 Start menu UI
* 🗂️ Interactive elements (click events, menus, etc.)
* 🎨 Smooth UI styling and transitions

---

## 🛠️ Built With

* HTML5
* CSS3 (Flexbox, Grid, Animations)
* JavaScript (Vanilla JS)
